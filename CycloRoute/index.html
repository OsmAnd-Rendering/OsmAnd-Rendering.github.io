CycloRoute

Un style de carte pour OsmAnd, axé sur la pratique du vélo et du cycloTourisme
(Dernière mise à jour 10/05/2024)



 

Caractéristiques du style :


  • Inspiré de CyclOSM :

    • Pistes cyclables plus lisibles et visibles
    • POI utiles / intéressants mis en évidence
    • Couleur opaque pour les itinéraires

  • Inspiré de Mapy.cz :

    • Couleur des terres et de l'eau

  • Réglage supplémentaire :

    • POI pour le cyclotourisme
    • Pour masquer les logos d'utilisation des terres

  • Encore plus ;-)

Instructions d'installation


  • Télécharger ce fichier

Première installation : Mise à jour du rendu :
  • Télécharger le fichier
  • Ouvrir avec OsmAnd
  • Cocher toutes les cases pour importer le Profil + rendu
  • Terminé !

    Votre nouveau profil, avec le rendu "CycloRoute", est prédéfini.
  • Télécharger le fichier
  • Ouvrir avec OsmAnd
  • Cocher uniquement la case "Ressources" pour n'importer que le rendu
  • Fait !

    Votre nouveau rendu "CycloRoute" a remplacé l'ancien
Si vous souhaitez l'utiliser pour le cyclotourisme :
  • Aller au réglage de la carte
    • Appuyer sur « Détails »
      • Activer le point d'intérêt pour le cyclotourisme

L'écran affichera les PDI utiles pour le Cyclotourisme en plus de ceux pour le vélo.
