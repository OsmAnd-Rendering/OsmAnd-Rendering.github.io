CycloRoute

A map style for OsmAnd, focused on the practice of cycling and cycloTouring
(Last update 10/05/2024)

 

Notable features over standard styles :


  • Inspired by CyclOSM :

    • More legible and visible cycle paths
    • Highlighted useful/interesting POIs
    • Opaque color for routes

  • Inspired by Mapy.cz :

    • Land and Water Color

  • Additional setting:

    • POI for cycle tourism
    • To hide land use logos

  • Even more ;-)

Instructions d'installation


  • Download this file
First installation : Update of the rendering :
  • Download the file
  • Open with OsmAnd
  • Check all the boxes to import the Profile + render
  • Done !

    Your new profile with the rendering is preset.
  • Download the file
  • Open with OsmAnd
  • Check only the "resources" box to import only the render
  • Done !

    Your new render has replaced the old one
If you want to use it for cyclotourism :
  • Go to map setting
    • Press "Details"
      • Enable POI for Cyclotouring

The screen n will only display useful POIs for Cyclotourism in addition to those for cycling.
